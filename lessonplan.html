Women's Health

Over the time spent on this project, i've used many skills, both recently acquired and some already developed. When coding a website from scratch, one must use their visualization skill to create a wireframe and picture the layout of the website. Once the layout is visualized, drawn and annotated, I used my basic HTML and CSS skills to create different sections of the site we call “divs”. Within these divs I am allowed to change the color, space between divs, the shape and size of each section through CSS. Throughout this project, i've learned to really take in mind the user experience of a site.
